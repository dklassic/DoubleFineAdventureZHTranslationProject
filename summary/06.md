# 06 — 尋找 Bagel 風格

- 團隊意識到原本美術流程不可行：Bagel 不可能同時負責所有概念、風格與最終繪製。
- Peter Chan 離開後，Levi Ryken 加入補足概念美術；Marius Fietzek 加入動畫與分鏡工作。
- Tim 完成更多關卡與謎題設計，包括 Cloud Colony 與 Spaceship Boy 的流程。
- Maiden’s Feast 暫時被放下，團隊轉向較能代表遊戲整體節奏的 Cloud Colony。
- Bagel 到辦公室後表明想參與構圖與概念，而不只是上色，促使團隊重新調整美術管線。

關鍵詞：Bagel、Levi Ryken、Marius Fietzek、Cloud Colony、Maiden’s Feast、Spaceship Boy。
