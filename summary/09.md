# 09 — 命名為 Broken Age

- Ron Gilbert 遊玩 Cloud Colony，肯定點擊冒險的寧靜感，也提出對話、移動速度、出口提示等問題。
- Ron 宣布離開 Double Fine，但承諾仍會遠端試玩並提供意見。
- Game Informer 即將來訪，團隊必須準備可展示的 Cloud Colony demo。
- 團隊製作 GDC 首支 teaser trailer，Marius 遠端製作 animatic，Brian McConnell 錄製音樂。
- 遊戲名稱經大量候選後定為《Broken Age》，象徵孩子們所處的破裂年代、尷尬年紀與打破循環。

關鍵詞：Ron Gilbert、Game Informer、GDC、《Broken Age》、Vella、Shay、teaser trailer。
